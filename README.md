A collection of my 3D printer configuration files, Klipper settings, peripheral configs, hardware notes, and reference material used for setup, troubleshooting, and upgrades.

## Overview

This repository is used to organize and document the hardware and software side of my 3D printer setup. It includes Klipper configuration files, board pin references, peripheral wiring notes, macros, and setup information for upgrades and accessories.

The goal is to keep everything version controlled, easy to reference, and easier to recover if I need to rebuild or troubleshoot the printer later.

## What’s in this repo

- Klipper configuration files
- Printer and MCU pinout references
- Peripheral configs and notes
- Macros for common printer actions
- Wiring and hardware documentation
- Upgrade notes and troubleshooting logs
- Reference links and setup notes from other sources

## Repository Contents

- Main `printer.cfg`
- Split Klipper config files
- Board and stepper driver pin mappings
- CR Touch / BLTouch notes
- Extruder and hotend config
- Fan wiring and control notes
- Display, camera, and sensor configs
- Macros such as pause, resume, and cancel print
- Notes for accessories and add-ons# Kender-3D-Printer
